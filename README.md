## מה יש בדשבורד

- 6 כרטיסי KPI (רווח יזמי, יח"ד, שטחי ציבור, חניה, מכפיל, הכנסות)
- טבלת השוואה: מצב קיים / מאושר / דרישת המועצה

### 7 שיעורי הכנה (עם הקראה קולית)
| # | שיעור | נושא |
|---|--------|-------|
| 1 | סיפור הפרויקט | רקע, מה קיים, מה מוצע, למה חשוב |
| 2 | הצדדים והשחקנים | מי מול מי, עו"ד, מומחים, עמדות |
| 3 | מה ביקשנו ומה קיבלנו | פערים: שטחי ציבור, חניה, חינוך, ניקוז |
| 4 | הרווח של היזם | 25.2% מול 16%, מצגת בורסה, תקן 21.1 |
| 5 | החניה והתחבורה | תקן 1:1.3 מול 1:1.5, רחוב הסביון, תח"צ |
| 6 | ההליכים המשפטיים | ציר זמן, ערר, עתירה, טענת אבן יהודה |
| 7 | הכנה אחרונה למו"מ | טקטיקה, סדר עדיפויות, תשובות מוכנות |

### פרקי עומק (אקורדיונים)
- **שטחי ציבור** - פרוגרמה, טעות בחישוב, חינוך, שצ"פ
- **רווח יזמי** - ניתוח 3 בעיות בדו"ח, מה אפשר "לקנות", תקן 21.1

- **ניקוז** - מתקנים פרטיים על קרקע ציבורית
- **גובה ומרחקים** - ניתוח + ויתור טקטי
- **השוואה לאבן יהודה** - טענת "איפה ואיפה"

### כלי מו"מ

- 4 שלבי טקטיקה

## איך לערוך

### שינוי סיסמה
חפשו בקובץ `index.html` את השורה:
```js
if (val === atob('U3BpdmFrMjAyNg==')) {
```
ה-`atob(...)` מפענח Base64. כדי לשנות סיסמה:
1. בחרו סיסמה חדשה
2. המירו ל-Base64 (בקונסולה של הדפדפן: `btoa('הסיסמה_החדשה')`)
3. החליפו את המחרוזת בתוך `atob('...')`

### עריכת שיעורים
חפשו בקובץ את:
```js
const lessons = [
```
כל שיעור הוא אובייקט עם:
```js
{
  icon: '&#128218;',        // אימוג'י
  color: '#3182ce',         // צבע כותרת
  title: 'שיעור 1: ...',   // כותרת
  sub: '~4 דקות קריאה',    // תיאור משני
  body: `...`               // תוכן HTML
}
```
אפשר להוסיף שיעורים חדשים (להוסיף אובייקט למערך) - וגם כפתור מתאים בסרגל השיעורים (חפשו `lessons-bar`).

### עריכת דרישות מו"מ
חפשו את ה-ID `demands` בקובץ. כל דרישה היא `div` עם class:
- `demand critical` - עדיפות קריטית (אדום)
- `demand high` - עדיפות גבוהה (כתום)
- `demand medium` - עדיפות בינונית (כחול)

### עריכת KPIs (מספרים למעלה)
חפשו `kpi-grid` - כל כרטיס הוא `div` עם class `kpi` + צבע (`red`, `green`, `orange`, `blue`).

### עריכת טבלאות
הטבלאות הן HTML רגיל (`<table>`). חפשו את הטקסט הרלוונטי ושנו.

### הוספת פרק חדש
1. הוסיפו לינק בסרגל הניווט (`<div class="nav">`)
2. הוסיפו `<div id="שם" class="section-anchor"></div>` במקום הרצוי
3. הוסיפו את תוכן הפרק ב-`<div class="card">`
